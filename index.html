Materi Pembelajaran SIG Berbasis Web

Dalam mata kuliah Praktikum SIG Berbasis Web, telah dipelajari berbagai Materi yang merupakan rincian dari komponen penyusun sebuah WebGIS. WebGIS merupakan teknologi yang mengabungkan aplikasi berbasis web dengan sistem informasi geografis untuk dapat berbagi, memvisualisasi, menganalisis, dan memanipulasi informasi geospasial melalui internet. WebGIS memungkinkan pengguna untuk mengakses dan berinteraksi dengan IG melalui browser, sehingga IG yang terdapat di dalamnya dapat diakses oleh berbagai pengguna di manapun.


Minggu 1: Cara Kerja Internet

Secara umum, internet bekerja dengan menggunakan tiga komponen utama, yaitu:

  1. Client: perangkat atau manusia yang melakukan akses terhadap website.
  2. Server: perangkat keras maupun lunak pada komputer yang berperan dalam menyimpan halaman web pada alamat tertentu.
  3. Bahasa atau protokol: media komunikasi yang menjembatani antara klien dan server.

Minggu 2: Pengenalan HTML dan CSS

Dalam pembuatan halaman web, terdapat tiga bahasa yang digunakan, dua diantaranya adalah HTML (HyperText Markup Language) dan CSS (Cascading Style Sheets).

  1. Bahasa HTML merupakan bahasa markup yang digunakan untuk membuat struktur website hingga menyusun format teks dan gambar pada sebuah halaman web.
  2. Bahasa CSS merupakan bahasa yang digunakan untuk berfungsi untuk mendesain, membentuk, serta mengubah tampilan halaman sebuah website.

Minggu 3: Pengenalan JavaScript

Selain HTML dan CSS, dalam pembuatan halaman web, juga digunkan bahasa lain yaitu JavaScript. JavaScript adalah bahasa pemrograman yang digunakan oleh pengembang untuk membuat halaman web yang interaktif. JavaScript mengatur behavior yang terdapat pada halaman web.

Minggu 4: Bootsrap Framework

Bootsrap Framework merupakan framework web development berbasis HTML, CSS, dan JavaScript yang dirancang untuk mempercepat proses pengembangan web, dengan adanya Bootsrap, developer tidak perlu menyusun sebuah web dari awal. Web yang dibangun dengan Bootstrap bersifat Responsive, artinya dapat dengan mudah menyesuaikan ukuran pada browser desktop atau mobile

Minggu 5 dan 6: Leaflet JS

LeafletJS merupakan library Open Source JavaScript yang digunakan dalam mempermudah deployment peta pada sebuah halaman web.

Minggu 7: Mobile GIS

Mobile GIS merupakan sebuah integrasi cara kerja perangkat lunak/keras untuk pengaksesan data dan layanan geospasial melalui perangkat bergerak via jaringan kabel atau nirkabel. Mobile GIS sangat bermanfaat, utamanya dalam kegiatan pengumpulan data geospasial, di mana nantinya akan terdapat dua perangkat yang digunakan dalam pengumpulan data tersebut, salah berperan untuk menginput data hasil pendataan di lapangan, dan perangkat lain berperan untuk menyimpan dan memvisualisasikan hasil pemetaan yang dilakukan.
